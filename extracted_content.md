<p>AI Survey Audit Prototype – Full Build Instructions for Antigravity</p><h2>1. Project Goal</h2><p>This prototype demonstrates an AI-powered survey auditing system. It allows a user to upload survey audio and form data, convert speech to text using Whisper, extract important information, compare it with form data, and detect mismatches.</p><h2>2. Complete System Flow</h2><p>1. User opens the web interface.<br />2. User fills survey form fields (name, age, location).<br />3. User uploads survey audio.<br />4. Frontend sends data to backend API.<br />5. Backend saves audio file temporarily.<br />6. Whisper converts audio to text.<br />7. Backend extracts key information.<br />8. Backend compares transcript with form data.<br />9. Backend stores the result in the database.<br />10. Backend returns transcript and match/mismatch result.<br />11. Frontend displays the result and flagged surveys.</p><h1>Complete Prototype Architecture</h1><p>Frontend (React + Tailwind)</p><p>        ↓</p><p>Backend API (FastAPI)</p><p>        ↓</p><p>Speech‑to‑Text Service (Whisper Local Model)</p><p>        ↓</p><p>Audit Logic (Mismatch Detection)</p><p>        ↓</p><p>Database Storage</p><p>        ↓</p><p>Audit Dashboard</p><h1>Folder Structure</h1><p>audit-ai-prototype/</p><p>│</p><p>├── backend/</p><p>│   ├── main.py</p><p>│   ├── whisper_service.py</p><p>│   ├── audit_logic.py</p><p>│   ├── database.py</p><p>│   ├── models.py</p><p>│   └── requirements.txt</p><p>│</p><p>├── frontend/</p><p>│   ├── src/</p><p>│   │   ├── pages/</p><p>│   │   │   ├── UploadSurvey.jsx</p><p>│   │   │   ├── ResultPage.jsx</p><p>│   │   │   └── Dashboard.jsx</p><p>│   │   ├── components/</p><p>│   │   │   ├── SurveyForm.jsx</p><p>│   │   │   └── ResultCard.jsx</p><p>│   │   └── App.jsx</p><p>│</p><p>└── database/</p><p>    └── survey.db</p><h1>Step 1 – Install Dependencies</h1><p>Install Python packages:</p><p>pip install fastapi</p><p>pip install uvicorn</p><p>pip install openai-whisper</p><p>pip install torch</p><p>pip install python-multipart</p><p>pip install sqlalchemy</p><p>Install Node.js dependencies for frontend:</p><p>npx create-react-app frontend</p><p>cd frontend</p><p>npm install axios</p><p>npm install tailwindcss</p><h1>Step 2 – Whisper Speech-to-Text Service</h1><p>Create whisper_service.py</p><p>Steps:</p><p>1.
